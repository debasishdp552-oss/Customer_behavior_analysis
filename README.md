# Customer_behavior_analysis
Data analytics project showcasing customer behavior analysis using python,sql and power bi

**Overview**

 This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The goal is to analyze customer shopping behavior, extract insights, and visualize key business metrics through an interactive dashboard.

**Dataset**

The dataset contains customer transaction and demographic information such as:

1.Age group

2.Gender

3.Purchase amount

4.Product category

5.Subscription status

6.Review rating

7.Payment method

8.Purchase frequency

**Tools & Technologies**

1.Python — data loading, cleaning, EDA

2.Pandas / NumPy — data manipulation

3.SQL (PostgreSQL via ODBC) — querying structured data

4.Aiven Cloud Database — remote database connection

5.Power BI — dashboard visualization

**Project Steps**

1.Loaded dataset into Python environment

2.Performed data cleaning and preprocessing

3.Conducted Exploratory Data Analysis (EDA)

4.Uploaded data to PostgreSQL database

5.Executed SQL queries for aggregated insights

6.Connected database to Power BI

7.Built interactive dashboard visuals

**Dashboard**

The Power BI dashboard presents:

1.Sales by category

2.Revenue by age group

3.Customer distribution

4.Subscription analysis

5.Payment method trends

It enables quick understanding of customer patterns and business performance.

**Results & Insights**

1.Identified highest revenue-generating customer segments

2.Detected purchasing trends across age groups

3.Compared category-wise sales performance

4.Analyzed impact of subscription status on spending

These insights can help businesses optimize marketing strategies and improve decision-making.

**How to Run This Project**

1.Clone repository

2.Install required Python libraries:
pip install pandas numpy sqlalchemy psycopg2

3.Configure database credentials

4.Run notebook/script step-by-step

5.Open Power BI file to view dashboard

**Conclusion**

This project showcases practical skills in data analysis, SQL querying, cloud database connectivity, and dashboard creation. It demonstrates the complete analytics pipeline from raw data to actionable insights.
